# Burnout check - Comprehensive Project Specifications (Student-Only Version)

เอกสารข้อมูลความต้องการระบบ (System Requirements) เกณฑ์การประเมินผลแบบ **Rule-Based** และโครงสร้างสถาปัตยกรรมข้อมูลสำหรับโปรเจกต์ **Burnout check** เพื่อใช้ในการพัฒนาและทดสอบระบบด้วย Next.js และ Vercel

---

## 1. Technical Stack & Deployment
- **Framework:** Next.js (App Router, Client-side processing only)
- **Styling:** Vanilla CSS & Tailwind CSS (Responsive Design, Custom Theme Tokens)
- **Deployment:** Vercel (Static / Frontend-only deployment)
- **Data Persistence:** `localStorage` (จัดเก็บข้อมูลบนเบราว์เซอร์ของผู้ใช้ ไม่ส่งผ่าน API ออกไปภายนอกเพื่อความปลอดภัยสูงสุดตามกฎหมาย PDPA)

---

## 2. Page Structure & Responsive UI Layouts

ระบบจะแบ่งออกเป็น 4 หน้าหลัก โดยมีการจัดวางองค์ประกอบให้รองรับแบบ Responsive (Mobile, Tablet, Desktop) ดังนี้:

### ก. หน้าหลัก (Landing Page - `/`)
- **Hero Section:** หัวเรื่องต้อนรับและปุ่ม Call to Action (CTA) เพื่อเข้าสู่การทำแบบประเมิน
- **Dimension Cards:** การ์ดอธิบาย 3 มิติหลักของการหมดไฟการเรียน (EE, CY, RE)
- **Privacy Assurance Callout:** กล่องข้อความแจ้งเตือนผู้ใช้งานเกี่ยวกับการจัดเก็บข้อมูลที่เป็นความลับและประมวลผลโลคัลเท่านั้น

### ข. หน้าทำแบบประเมิน (Assessment Form - `/assessment`)
- **Responsive Stepper:** 
  - *Desktop/Tablet:* แถบเส้นเชื่อมต่อแบบสเต็ปเชิงเส้น (Linear Stepper) 1 -> 2 -> 3 -> 4 แสดงชื่อหมวดหมู่ย่อย และแสดงขีดติ๊กถูกสีขาวเมื่อเสร็จสิ้นแต่ละขั้นตอน
  - *Mobile:* การ์ดหัวข้อขั้นตอนปัจจุบัน (เช่น "ขั้นตอนที่ 1 จาก 4") พร้อมแถบความคืบหน้า (Progress Bar) และตัวเลขเปอร์เซ็นต์
- **Step-by-step Sections:**
  - **ขั้นตอนที่ 1:** ข้อมูลทั่วไปและพฤติกรรมการใช้ชีวิต (Demographics & Lifestyle)
  - **ขั้นตอนที่ 2:** แบบประเมินมิติความล้าทางอารมณ์และร่างกาย (Emotional Exhaustion - 9 ข้อ)
  - **ขั้นตอนที่ 3:** แบบประเมินมิติการเมินเฉยต่องานและการเรียน (Cynicism - 9 ข้อ)
  - **ขั้นตอนที่ 4:** แบบประเมินมิติความรู้สึกไม่ประสบความสำเร็จ (Reduced Efficacy - 9 ข้อ)
- **ระบบตรวจสอบคำตอบและนำทางอัตโนมัติ (Validation & Auto-scroll):**
  - เมื่อผู้ใช้กดปุ่ม "ถัดไป" หรือ "ส่งแบบประเมิน" ระบบจะสแกนหาข้อที่ยังไม่ได้ตอบ
  - กล่องคำถามที่ถูกข้ามจะแสดงกรอบและพื้นหลังเป็น **สีแดง** (`border-error bg-error-container/5 ring-1 ring-error`)
  - หน้าจอจะทำการเลื่อนหน้าจออัตโนมัติ (**Smooth Auto-scroll**) ไปที่ข้อแรกที่ยังไม่ได้ตอบทันที

### ค. หน้าแสดงผลลัพธ์การคัดกรอง (Results UI - `/results`)
- **Bento Grid Layout:**
  - **Risk Status Card:** ระดับความเสี่ยง (สูง/ปานกลาง/ต่ำ) พร้อมป้ายชื่อระดับชั้นและแผนการเรียนของนักเรียน
  - **Burnout Risk Gauge:** เกจ์วัดความเสี่ยงครึ่งวงกลมที่มีเข็มชี้วัดหมุนตามระดับเปอร์เซ็นต์ความเสี่ยง
  - **Radar Chart Card:** กราฟ SVG เรดาร์ 3 มิติแจกแจงคะแนน EE, CY, RE
  - **Key Factor Cards:** ปรับตามกฎวิเคราะห์เพื่อดึงปัจจัยปัญหาลบสะสมสูงสุด 2 อันดับแรกขึ้นมาโชว์
  - **Self-Tracking History Line:** ไทม์ไลน์ประวัติการคัดกรอง 5 ครั้งล่าสุด
  - **Scoring Table:** ตารางระบุข้อคำถาม คำตอบ และคะแนนคัดกรองที่ได้ในแต่ละข้อแบบละเอียด
- **Counselor Report Format (`@media print`):** เทมเพลต CSS สั่งพิมพ์สำหรับครูแนะแนว โดยซ่อนปุ่มนำทาง และปรับเลย์เอาต์ให้อยู่ในกระดาษ A4 หน้าเดียวอย่างลงตัว

### ง. หน้าคู่มือดูแลตนเอง (Coping Recommendations - `/recommendations`)
- **Back to Results Link:** ปุ่มนำทางกลับไปหน้าผลลัพธ์ที่ด้านบนสุด
- **Symmetrical Mobile Tab Bar:** เมนูเลือกหัวข้อการผ่อนคลายความเครียดที่ปรับเป็นปุ่มการ์ดกริดสมมาตร 2x2 บนมือถือ และแถบแท็บยาวบนจอคอมพิวเตอร์
- **Help Resource Directory:** ช่องทางสายด่วนและข้อแนะนำการขอรับความช่วยเหลือจากครูแนะแนวประจำโรงเรียน

## 3. Complete Questionnaire Text (ข้อคำถามการประเมิน 27 ข้อ)

ตัวเลือกคำตอบอ้างอิงระดับความถี่พฤติกรรม (มาตราส่วน Likert Scale 1-5 คะแนน):
1. **ไม่เลย** = 1 คะแนน
2. **นานๆ ครั้ง** = 2 คะแนน
3. **บางครั้ง** = 3 คะแนน
4. **บ่อย** = 4 คะแนน
5. **บ่อยมาก** = 5 คะแนน

### ส่วนที่ 1: ด้านความอ่อนล้าทางอารมณ์และร่างกาย (Emotional Exhaustion: EE)
1. ฉันรู้สึกห่อเหี่ยวจิตใจ หมดเรี่ยวแรง และสูญเสียแรงจูงใจในการทำงาน/การเรียน
2. ฉันรู้สึกอ่อนเพลียตั้งแต่ตื่นนอน เมื่อคิดว่าต้องไปเผชิญกับงาน/การเรียนในวันนี้
3. ฉันรู้สึกเครียด ท้อแท้ และรู้สึกว่าตนเองกำลังรับภาระหนักเกินไปจนจัดการไม่ไหว
4. ฉันมีอาการทางกายที่ผิดปกติ เช่น ปวดศีรษะ ปวดกล้ามเนื้อ หรือนอนไม่หลับเรื้อรัง
5. ฉันรู้สึกหมดความอดทน หงุดหงิด โกรธง่าย และอารมณ์แปรปรวนได้ง่ายกว่าปกติ
6. ฉันรู้สึกหมดเรี่ยวแรงเมื่อสิ้นสุดวันจากการเรียน การทำโครงงาน หรือการอ่านหนังสือ
7. แค่ตื่นขึ้นมาแล้วคิดว่าต้องไปเรียน หรือต้องเผชิญกับการเตรียมตัวสอบ ฉันก็รู้สึกเหนื่อยล้าแล้ว
8. การต้องทำความเข้าใจเนื้อหา หรือแก้โจทย์ปัญหาซับซ้อนตลอดทั้งวัน ทำให้ฉันรู้สึกเครียดมาก
9. ฉันรู้สึกว่าตนเองกำลังรับภาระความคาดหวัง (ทั้งจากตัวเองและคนรอบข้าง) ที่หนักเกินไป

### ส่วนที่ 2: ด้านการเมินเฉยและการลดค่าความเป็นบุคคล (Cynicism / Depersonalization: CY)
1. ฉันเริ่มมีทัศนคติในแง่ลบต่องาน/การเรียน และมองคนรอบข้างหรือสิ่งแวดล้อมในแง่ร้าย
2. ฉันรู้สึกอยากแยกตัวออกห่าง ไม่ต้องการสุงสิงกับเพื่อนร่วมงาน เพื่อนร่วมชั้น หรือผู้รับบริการ
3. ฉันปฏิบัติต่อผู้อื่นอย่างแข็งกระด้าง หรือรู้สึกว่าตนเองไร้ความเห็นอกเห็นใจผู้อื่น
4. ฉันรู้สึกเมินเฉย ถอดใจ และไม่อยากใส่ใจต่อพฤติกรรมของคนรอบข้างหรือผลลัพธ์ของงานอีกต่อไป
5. ฉันรู้สึกหวาดระแวง คอยจับผิด โทษคนอื่น หรือคิดว่าปัญหาที่เกิดขึ้นเป็นความผิดของผู้อื่น
6. ฉันเริ่มรู้สึกว่าสิ่งที่กำลังพยายามศึกษาหรือเตรียมตัวอยู่ ไม่มีประโยชน์หรือไม่มีความหมาย
7. ฉันเริ่มตีตัวออกห่าง ไม่อยากพูดคุยเรื่องการเรียน เรื่องสอบ หรือเรื่องอนาคตกะบเพื่อนและครอบครัว
8. ฉันรู้สึกเบื่อหน่ายและสูญเสียความกระตือรือร้นในการทำให้เป้าหมายสำเร็จอย่างสิ้นเชิง
9. ฉันเริ่มมีทัศนคติในแง่ลบและหงุดหงิดง่ายกับระบบการศึกษา หรือสภาพแวดล้อมรอบตัว

### ส่วนที่ 3: ด้านความรู้สึกไม่ประสบความสำเร็จและประสิทธิภาพลดลง (Reduced Efficacy: RE)
1. ฉันรู้สึกว่าตนเองไม่มีความสามารถ ไร้ประสิทธิภาพในการแก้ปัญหาและจัดการงาน/การเรียน
2. ฉันรู้สึกว่าสิ่งที่ฉันทำอยู่ไม่มีคุณค่า ไม่มีความก้าวหน้า และมองไม่เห็นความสำเร็จ
3. ฉันขาดความคิดริเริ่มสร้างสรรค์ในการทำสิ่งใหม่ๆ และคุณภาพงาน/ผลการเรียนของฉันลดลงอย่างเห็นได้ชัด
4. ฉันรู้สึกสงสัยในความสามารถของตนเอง สูญเสียความมั่นใจ และรู้สึกว่าตนเองล้มเหลว
5. ฉันรู้สึกอยากหลีกเลี่ยงปัญหา มีการขาดงาน/ขาดเรียนบ่อยขึ้น หรือมีความคิดที่อยากจะลาออก
6. ฉันสามารถจัดการกับปัญหา หรือโจทย์ยากๆ ที่เกิดขึ้นในการเรียนได้เป็นอย่างดี *(คำถามเชิงบวก - คิดคะแนนกลับ)*
7. ฉันรู้สึกว่าตนเองได้เรียนรู้สิ่งใหม่ๆ และพัฒนาความสามารถของตนเองอย่างต่อเนื่อง *(คำถามเชิงบวก - คิดคะแนนกลับ)*
8. ฉันมั่นใจว่าตนเองจะสามารถทำเป้าหมายให้สำเร็จได้ตามที่ตั้งใจไว้ *(คำถามเชิงบวก - คิดคะแนนกลับ)*
9. ฉันรู้สึกภูมิใจกับผลงานและสิ่งที่ตนเองทำได้ในช่วงที่ผ่านมา *(คำถามเชิงบวก - คิดคะแนนกลับ)*

## 4. Rule-Based Scoring Engine & Visualization Math

### ก. การคำนวณคะแนนเฉลี่ยแต่ละด้าน
- **Emotional Exhaustion Score (EE):** 
  $$EE = \frac{\sum_{i=1}^{9} EE_i}{9}$$
- **Cynicism Score (CY):** 
  $$CY = \frac{\sum_{i=1}^{9} CY_i}{9}$$
- **Reduced Efficacy Score (RE):** 
  เนื่องจากข้อ 6-9 เป็นคำถามเชิงบวก ระบบจะทำการสลับเกณฑ์คำตอบ (Reverse Scoring) ก่อนการเฉลี่ยสะสม:
  $$RE_i' = 6 - RE_i \quad (\text{สำหรับ } i \in \{6, 7, 8, 9\})$$
  $$RE = \frac{\sum_{i=1}^{5} RE_i + \sum_{j=6}^{9} RE_j'}{9}$$

### ข. การประเมินเปอร์เซ็นต์ความเสี่ยงรวม (Overall Risk %)
- **Weighted Overall Score:** 
  $$\text{Overall Score} = (EE \times 0.45) + (CY \times 0.35) + (RE \times 0.20)$$
- **Overall Risk Percentage:** แปลงระดับคะแนน (1.00 - 5.00) ให้เป็นเปอร์เซ็นต์ความเสี่ยง (0% - 100%):
  $$\text{Risk Percentage} = \text{round}\left(\frac{\text{Overall Score} - 1.0}{4.0} \times 100\right)$$
- **การจัดกลุ่มเกณฑ์ความเสี่ยง (Risk Level):**
  - สูง (High Risk): $\ge 70\%$
  - ปานกลาง (Moderate Risk): $\ge 35\%$ และ $< 70\%$
  - ต่ำ (Low Risk): $< 35\%$

### ค. คณิตศาสตร์สำหรับการจำลองภาพด้วย SVG (Visualization Math)
1. **เข็มเกจ์วัดความเสี่ยง (Risk Gauge Needle):**
   เกจ์รูปครึ่งวงกลม มีพิกัดการหมุนเข็มเริ่มต้นที่ $-90^\circ$ (ซ้ายสุด - 0%) ถึง $+90^\circ$ (ขวาสุด - 100%):
   $$\theta = -90 + \left(\frac{\text{Risk Percentage}}{100} \times 180\right)$$
   ใช้สไตล์ CSS: `transform: rotate(\theta deg)`

2. **กราฟเรดาร์ 3 มิติ (SVG 3-Dimension Radar Chart):**
   จุดศูนย์กลางของแผนภูมิ SVG อยู่ที่พิกัด $(50, 50)$ โดยมีรัศมีสูงสุดสำหรับการแสดงผลคะแนนเท่ากับ $40$ พิกเซล การคิดค่าอัตราส่วนคะแนนอิงจากสูตร $Score_{norm} = \frac{Score - 1}{4}$
   - **ความล้าอารมณ์ (EE) - ชี้ไปทิศเหนือ ($90^\circ$ ทวนเข็ม):**
     $$p_1 = (50, 50 - (EE_{norm} \times 40))$$
   - **การเมินเฉย (CY) - ชี้ไปทิศตะวันออกเฉียงใต้ ($30^\circ$ ตามเข็มด้านล่าง):**
     $$p_2 = (50 + (CY_{norm} \times 40 \times \cos(30^\circ)), 50 + (CY_{norm} \times 40 \times \sin(30^\circ)))$$
     *(โดยค่าประมาณ $\cos(30^\circ) \approx 0.866$ และ $\sin(30^\circ) = 0.5$)*
   - **ความรู้สึกไม่สำเร็จ (RE) - ชี้ไปทิศตะวันตกเฉียงใต้ ($150^\circ$ ตามเข็มด้านล่าง):**
     $$p_3 = (50 - (RE_{norm} \times 40 \times \cos(30^\circ)), 50 + (RE_{norm} \times 40 \times \sin(30^\circ)))$$
   
   *วาดขอบเขตกราฟด้วยแท็ก SVG `<polygon points="p1_x,p1_y p2_x,p2_y p3_x,p3_y" />`*

## 5. Lifestyle Factor Trigger Rules (กฎวิเคราะห์หาปัจจัยหลักที่เป็นปัญหาสุด 2 อันดับแรก)

ระบบจะทำการคัดกรองพฤติกรรมและการเรียนในสเต็ปที่ 1 รวมทั้งคะแนนมิติอารมณ์เฉลี่ย เพื่อระบุปัจจัยที่มีปัญหาที่สุดขึ้นมาโชว์ 2 รายการตามกฎเกณฑ์ดังนี้:

| ปัจจัย (Factor) | ตัวแปรคีย์ | เงื่อนไขการกระตุ้น (Trigger Rule) | ข้อความแสดงผลบนหน้าจอ |
| :--- | :--- | :--- | :--- |
| **การนอนหลับ** | `sleep` | `sleep < 5.0` | **พักผ่อนไม่เพียงพอรุนแรง:** การนอนหลับน้อยกว่า 5 ชม. สะสม ส่งผลต่ออารมณ์และการจดจ่อ |
| **ภาระงาน** | `workload` | `workload >= 4` | **ภาระงานสะสมสูง:** ปริมาณการบ้านและงานค้างสร้างความกดดันทางใจ |
| **ภาวะอารมณ์ล้า** | `exhaustion` | `EE >= 3.8` | **ความเหนื่อยล้าทางอารมณ์สูง:** รู้สึกหมดพลัง ขาดพลังใจในการเรียนและการทำกิจวัตรประจำวัน |
| **กิจกรรมผ่อนคลาย** | `hobbies` | `hobbies <= 2` | **ขาดกิจกรรมผ่อนคลาย:** ไม่มีพื้นที่ระบายความเครียด ทำให้อารมณ์ด้านลบสะสม |
| **ทัศนคติลบ** | `cynicism` | `CY >= 3.5` | **ทัศนคติลบและห่างเหินการเรียน:** เริ่มรู้สึกการเรียนไร้เป้าหมายหรือหมดแรงบันดาลใจ |
| **ขาดสังคมช่วยเหลือ** | `socialSupport` | `socialSupport <= 2` | **ระบบช่วยเหลือรอบข้างจำกัด:** ขาดการระบายความรู้สึกและความตึงเครียดกับคนรอบข้างหรือครอบครัว |
| **ปฏิกิริยาทางกาย** | `physicalSymptoms` | `physicalSymptoms >= 3` | **ปฏิกิริยาตึงเครียดออกทางกาย:** ความเครียดสะสมเริ่มก่อตัวเป็นอาการปวดหัว ปวดบ่าไหล่ หรือรบกวนการนอนหลับ |
| **เวลาหน้าจอ** | `screen` | `screenTime > 8` | **ใช้เวลาหน้าจอสูงมาก:** การใช้งานมือถือ/คอมฯ นานเกิน 8 ชม. ต่อวันกระตุ้นความล้าทางสายตาและสมอง |
| **โภชนาการ** | `nutrition` | `nutrition == "no"` | **รับประทานอาหารไม่สมดุล:** ทานอาหารไม่ตรงเวลาหรือโภชนาการที่ไม่เหมาะสม |

### กฎการเรียงลำดับความสำคัญ (Priority Selection Sorting Order)
หากผู้เรียนผ่านเกณฑ์การกระตุ้นข้างต้นพร้อมกันมากกว่า 2 ข้อ ระบบจะเรียงคีย์จัดลำดับเพื่อเลือก 2 ข้อแรกมาแสดงตามลำดับความสำคัญดังนี้:
`sleep` -> `workload` -> `exhaustion` -> `hobbies` -> `cynicism` -> `socialSupport` -> `physicalSymptoms` -> `screen` -> `nutrition`

## 6. Dynamic Insights & Personalized Action Plans

### ก. การแสดงผลบทวิเคราะห์เชิงลึกรวม (Overall Insights)
- **เสี่ยงสูง (High Risk):** *"จากการวิเคราะห์พฤติกรรมและความรู้สึกของคุณ ระบบตรวจพบระดับความเสี่ยงหมดไฟในระดับวิกฤต ซึ่งอาจส่งผลกระทบต่อทั้งการเรียนและสภาพจิตใจโดยตรง แนะนำให้หยุดพักฟื้นฟูจิตใจ พูดคุยกับครูแนะแนวที่โรงเรียน หรือติดต่อสายด่วนสุขภาพจิต 1323 เพื่อวางแผนปรับภาระงานและรับคำแนะนำที่ถูกต้อง"*
- **เสี่ยงปานกลาง (Moderate Risk):** *"ระดับความเสี่ยงอยู่ในเกณฑ์เริ่มเฝ้าระวัง มีความอ่อนล้าทางใจสะสมเนื่องจากสมดุลวิถีชีวิตเสียไป แนะนำให้ตั้งเป้าหมายลดชั่วโมงหน้าจอและเพิ่มการนอนหลับให้ได้อย่างน้อย 6-7 ชม. รวมทั้งแบ่งเวลาทำงานเป็นช่วง ๆ (เช่น Pomodoro) เพื่อลดแรงกดดัน"*
- **เสี่ยงต่ำ (Low Risk):** *"คุณรักษาสมดุลชีวิตและจิตใจได้ดีมาก ระดับความเสี่ยงหมดไฟอยู่ในเกณฑ์ปกติ แนะนำให้คงพฤติกรรมการนอนและพักผ่อนในเกณฑ์ที่ดีแบบนี้ต่อไป เพื่อสร้างภูมิคุ้มกันความเครียดในระยะยาว"*

### ข. คำแนะนำแบบเฉพาะเจาะจงรายมิติ (Personalized Action Plan)
แนะนำโดยวิเคราะห์หาด้านที่มีคะแนนเฉลี่ยสูงสุดใน 3 มิติ (EE, CY, RE):
- **EE สูงสุด:** จัดสรรเวลาพักผ่อนเชิงรุก (Proactive Rest) เช่น เวลานอนที่สม่ำเสมออย่างน้อย 7-8 ชม., ทำกิจกรรมผ่อนคลายโดยงดใช้หน้าจอ (Digital Detox)
- **CY สูงสุด:** หาเวลาแชร์ปัญหาความรู้สึกร่วมกับผู้รอบข้าง (ครอบครัว, เพื่อนสนิท, หรือครูแนะแนว) และทำกิจกรรมที่สลับเปลี่ยนสภาพแวดล้อมจำเจเพื่อฟื้นฟูแรงบันดาลใจ
- **RE สูงสุด:** ประยุกต์ใช้เทคนิคซอยเป้าหมายการเรียนหรืองานออกเป็นส่วนเล็กๆ (Micro-tasks) ที่พิชิตได้ง่ายใน 15 นาที และฉลองเป้าหมายย่อยนั้นเพื่อฟื้นฟูความมั่นใจ
